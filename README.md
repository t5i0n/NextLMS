# NextLMS

A full-stack Learning Management System (LMS) designed to simulate a modern e-learning platform. It allows instructors to create and manage courses while students can enroll, learn, and track their progress.

---

## Features

- User Authentication (Login / Register)
- Role-Based Access (Instructor / Student)
- Course Creation & Management
- Student Enrollment
- Progress Tracking
- Responsive User Interface

---
